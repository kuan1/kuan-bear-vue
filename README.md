# kuan-bear-vue （vue 项目整理）

> 针对幻熊公司项目使用

## 使用

```
import Vue from 'vue'
import BearVue from 'kuan-bear-vue'

// 只会自动注入directives和filter
vue.use(BearVue)
```

## 组件

```
Indicator // loading
Toast // 提示
MyCode // 二维码
MyVideo // 视频
HeadRoom // 头部
```
